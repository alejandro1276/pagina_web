Sistemas Operativos

Un Sistema operativo es un programa que administra el hardware de una computadora. Tambien proporciona las bases para los programas de aplicacion, y actua como intermediario entre el usuario y el hardware. Estas tareas, pueden ser llevadas a cabo de varias formas, lo que permite que algunos sistemas Operativos se diseñen para ser practicos, otros eficientes y otros para ser ambas cosas.
